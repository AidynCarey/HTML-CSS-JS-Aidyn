Classes

COP4813

This course focuses on the principles and technologies used to build modern, data-driven websites and applications. Topics include HTML, CSS, JavaScript, server-side programming, and database integration. Students learn how to create responsive, standards-compliant web pages, work with APIs, and develop secure, maintainable web solutions. The class emphasizes practical, hands-on projects to demonstrate web development concepts and best practices.

CEN4801

Systems Integration explores how diverse technologies, platforms, and software applications can be combined into a single, functional solution. The course covers systems architecture, middleware, APIs, and enterprise integration patterns. Students gain experience in analyzing requirements, designing integration strategies, and implementing solutions that enable different systems to communicate efficiently. Real-world case studies highlight issues such as scalability, performance, and security in enterprise environments.

CDA4101

This course introduces the internal structure and operation of modern computer systems. Students study topics such as digital logic, processor architecture, instruction sets, memory hierarchy, and input/output mechanisms. The class emphasizes how software interacts with hardware, providing insight into performance optimization, assembly language programming, and the trade-offs in computer system design. The course provides a foundation for understanding how computers execute programs at the hardware level.
